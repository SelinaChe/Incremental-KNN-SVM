# Incremental-KNN-SVM
基本思想：使用KNN与SVM的混合算法，在一定的邻域内完成SVM的多分类，在KNN的基础上完成增量与增类，并使用kd-tree为IKNN-SVM算法提速。



paper: IKNN-SVM: A Hybrid Incremental Algorithm for Image Classification
